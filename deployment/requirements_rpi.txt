# ----------------------------------------------------------------------------
# Platform Specific Notes
# ----------------------------------------------------------------------------
# 
# For Raspberry Pi 5 with Bookworm OS (64-bit):
# 
# 1. System packages required before pip install:
#    sudo apt install -y python3-pip python3-venv python3-dev
#    sudo apt install -y libportaudio2 portaudio19-dev
#    sudo apt install -y libatlas-base-dev libopenblas-dev
#    sudo apt install -y libjpeg-dev libtiff-dev libpng-dev
#    sudo apt install -y libavcodec-dev libavformat-dev libswscale-dev
#    sudo apt install -y python3-picamera2
#
# 2. PyTorch installation notes:
#    - Use CPU-only version (no CUDA on Raspberry Pi)
#    - torch 2.0.0 is tested and stable on RPi 5
#    - If installation fails, try: pip install torch --extra-index-url https://download.pytorch.org/whl/cpu
#
# 3. OpenCV installation notes:
#    - opencv-python-headless is lighter (no GUI)
#    - If cv2.imshow() needed, use opencv-python instead
#
# 4. PyAudio installation notes:
#    - Requires portaudio19-dev system package
#    - If pip install fails, try: sudo apt install python3-pyaudio
#
# 5. Memory considerations:
#    - Raspberry Pi 5 with 4GB RAM is minimum
#    - 8GB RAM recommended for better performance
#    - Consider swap space if RAM is limited
#
# 6. Performance optimization:
#    - Use virtual environment to avoid conflicts
#    - Install only required packages in production
#    - Comment out development dependencies
#
